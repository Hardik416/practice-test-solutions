# 🚀 Practice Test Solutions (C++)

Welcome! This repository contains **C++ solutions for practice tests provided by Chitkara University** 🏫

It is designed for quick revision, coding practice, and improving problem-solving skills in a clean developer-friendly format 💻

---

## 🧰 Tech Stack

- **Language:** C++ (100%) ⚙️
- **Standard:** C++17 (recommended) ✅

Solutions are organized across multiple practice test sets, such as:

- 📝 `PT4`
- 📝 `PT5`

> Each file is intended to be easy to run and easy to understand 🔍

---

## ▶️ Build & Run

Compile any C++ file:

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic <file>.cpp -o app
```

Run:

```bash
./app
```
